# 关于项目

## 为什么写这个项目？
该项目为本人在2017年暑期通过html、Bootstarp、css3、js写的GTA游戏主题网页！现和另外两名前端小伙伴使用React进行重构！同时对毕业以来所学的知识进行巩固！如果该项目有帮助到你学习或理解react，欢迎Star!
